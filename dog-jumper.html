Dog Jumpers

For the dog jumpers, I am currently still purchasing fabrics on request, which makes that you can be very creative in what you would like. I also will add some inspiration below the form, so if you like to see that, please click here.

In order to make a custom size dog jumper, I will need the following measurements:

  1. The back measurement:
    • From the neck crease (where the collar is placed) to the tail bone / where the tail starts.
  2. The front leg length.
  3. Chest measurement:
    • This should be the widest part of the ribcage.
  4. Hip measurement:
    • This should be the slimmest part around the hips.
